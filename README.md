# vbook_Klein

Repository index cho VBook theo format raw GitHub.

## Cau truc

- `plugin.json`: file index tong hop extension
- `storya/plugin.zip`: goi extension de app VBook tai ve
- `storya/plugin.json`: metadata/script cua extension
- `storya/icon.png`: icon extension

## Raw URL can dung

- `https://raw.githubusercontent.com/chillguyk5/vbook_Klein/main/plugin.json`

## Cach cap nhat version

1. Sua source trong `storya/src/*` (neu can) tu repo dev.
2. Build lai `storya/plugin.zip`.
3. Tang `version` trong `storya/plugin.json` va `plugin.json` root.
4. Commit/push len nhanh `main`.
